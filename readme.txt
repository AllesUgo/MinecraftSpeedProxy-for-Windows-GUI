int __cdecl StartServer(const char *serveraddr, int remoteport, int localport)
启动服务器参数，参数列表:	服务器地址	服务器端口	本地端口
返回值：	0成功		非零失败
说明：
启动服务器，开始监听端口并允许连接，第一次运行本函数时会初始化网络和加载motd，之后关闭服务器后重新运行（不是退出进程）不会重新初始化
网络和motd。若在服务器没有退出前再次调用本函数会返回1

int __cdecl StopServer()
停止监听端口 参数列表：空
返回值	0成功		非零失败
说明：
使用本函数来关闭服务器，本函数会停止等待连接函数，但不会断开已有客户端连接
若在服务器没有启动前调用会返回-1，本函数会造成阻塞直到服务线程退出（并非客户端转发线程），一般速度很快
